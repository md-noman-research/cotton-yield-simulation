## A note before you use this

Everything generated by this codebase comes from stochastic simulation — there are no field trials, no soil samples, no experimental plots. The numbers are model outputs, calibrated against established Maas–Hoffman salinity thresholds. If you are looking for empirical cotton yield data, this is not that. What this *is*, is a robust computational pre-screening framework with quantified uncertainty, designed to explore species resilience under non-stationary coastal salinity.

---

## What this study is about

Coastal salinity in southern Bangladesh is a dynamic threat, especially during the pre-monsoon period where capillary salt accumulation peaks. This model compares the resilience of two ecologically distinct cotton species: 
- *Gossypium barbadense* (Suvin) — an extra-long-staple tetraploid.
- *Gossypium arboreum* (Muslin) — a short-staple diploid with centuries of localized adaptation.

The simulation runs 500 Monte Carlo realizations per species. It utilizes phenologically weighted Maas–Hoffman kinetics (weighting the boll-set phase 3× higher) and a synergistic stress interaction term (1.5× penalty coefficient) based on the synergistic stress response concepts for Malvaceae (Mittler, 2006).

---

## New in Version 1.1: Robustness & Scenarios

To address the inherent uncertainty in unvalidated literature coefficients (OAM, fiber degradation rates), this repository now includes:

1.  **Stress Regimes**: You can now simulate specific climate scenarios, moving beyond the "salinity-only" baseline to explore heat waves and combined extremes where stressors sync.
2.  **Global Sensitivity Analysis**: Using Sobol indices and Tornado plots, the model quantifies exactly how much of the yield variance is driven by parameter uncertainty versus environmental stochasticity.

## Getting started

```bash
git clone https://github.com/md-noman-research/cotton-yield-simulation.git
cd cotton-yield-simulation
pip install -r requirements.txt
```

### 1. Run the Sensitivity Analysis (Recommended First Step)
Before trusting the mean projections, quantify the uncertainty:
```bash
python sensitivity_analysis.py
```
Outputs Tornado plots and Sobol indices to `./sensitivity_results/`.

### 2. Run Simulations under Stress Scenarios
Simulate a combined heat wave and extreme salinity peak:
```bash
python stochastic_simulation_model.py --stress-regime combined-extreme
```

### 3. Generate Publication Figures
```bash
python generate_figures.py
```

## Reproducibility & Validation

- **Integrity**: Every run generates a `integrity_log.json` with SHA-256 hashes of the CSV data.
- **Sobol S1/ST**: Sensitivity indices confirm that the directional trend of Muslin's stability advantage is robust to a ±17% uncertainty range in the OAM coefficient.
- **Interaction Logic**: The 1.5× synergy factor is justified by Mittler (2006) for co-occurring abiotic stressors.
