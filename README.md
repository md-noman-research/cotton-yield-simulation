## A note before you use this

Everything generated by this codebase comes from stochastic simulation — there are no field trials, no soil samples, no experimental plots. The numbers are model outputs, calibrated against established Maas–Hoffman salinity thresholds. If you are looking for empirical cotton yield data, this is not that. What this *is*, is a pre-screening model designed to explore how two quite different cotton species might behave under the kind of variable coastal salinity conditions we see in the Noakhali region of Bangladesh, before anyone commits resources to a full field study.

---

## What this study is about

Coastal salinity in southern Bangladesh has been getting worse. The pre-monsoon period is especially severe — capillary salt accumulation can reach levels that cotton simply cannot tolerate. At the same time, there are two cotton species with very different salt-handling strategies: *Gossypium barbadense* (the Suvin variety, an extra-long-staple type) and *Gossypium arboreum* (Muslin, a short-staple diploid that has been grown in Bengal for centuries). The question this model tries to answer is: under realistic, stochastic salinity conditions, how differently do they actually perform — in terms of yield, but also fiber quality?

The simulation runs 500 Monte Carlo realizations per species. Salinity follows a non-stationary AR(1) process that mimics the seasonal capillary rise and monsoon washout pattern from SRDI's 2020 Noakhali coastal survey. Temperature follows its own AR(1) process. Yield loss is calculated using Maas–Hoffman kinetics, weighted by growth phase (boll set gets three times the weight of vegetative or maturation phases, because that is genuinely when salt stress does the most damage). Fiber quality degrades linearly above the salinity threshold, based on rates from Pettigrew (2004) and Francois et al. (1994).

## Species parameters at a glance

Both species share the same Maas–Hoffman threshold (7.7 dS/m) and slope (5.2%/dS/m), which are the *G. hirsutum* proxy values from the literature — this is a deliberate modeling choice discussed in the manuscript. What differs is the Osmotic Adjustment Modifier: Suvin carries an OAM of 1.00 (no assumed buffering beyond the standard model), while Muslin carries 0.60, reflecting the 40% osmotic stress reduction reported by Sharif et al. (2019) for *G. arboreum*.

| | Suvin (*G. barbadense*) | Muslin (*G. arboreum*) |
|---|---|---|
| Max yield (kg/ha) | 1 800 | 1 200 |
| OAM | 1.00 | 0.60 |
| Salinity threshold (dS/m) | 7.7 | 7.7 |
| Staple baseline (mm) | 33.2 | 18.5 |
| Micronaire baseline (µg/inch) | 3.70 | 4.60 |
| Bundle strength baseline (g/tex) | 31.5 | 23.8 |

## Getting started

```bash
git clone https://github.com/md-noman-research/cotton-yield-simulation.git
cd cotton-yield-simulation
pip install -r requirements.txt
```

Python 3.10 or later is required. The `statsmodels`, `scikit-learn`, and `scipy` versions in `requirements.txt` matter for exact numerical reproducibility.

**Run the simulation:**
```bash
python stochastic_simulation_model.py
```

That writes five CSVs to `./simulation_outputs/` — the two species run files, the Maas–Hoffman validation table, the fiber quality lookup table, and an aggregate summary. It also writes a `integrity_log.json` with SHA-256 hashes of every output, so anyone can verify they got the same numbers.

**Generate the figures:**
```bash
python generate_figures.py
```

Five publication-ready figures go to `./figures/`.

**Optional arguments for the simulation:**
```bash
python stochastic_simulation_model.py --n-runs 1000 --seed 7 --output-dir ./my_run
```

## Reproducibility

The default seed is 42 with 500 runs over a 120-day season. Each Monte Carlo realization uses its own seed offset so the salinity and temperature trajectories are independent across runs but fully deterministic given the same master seed. Changing `--seed` will change every output number; the integrity log will reflect that.

## References this model is built on

- Maas & Hoffman (1977) — the foundational threshold-slope salinity damage model
- Pettigrew (2004, *Agronomy Journal*) — fiber quality degradation rates under salinity
- Ashraf (2002, *Critical Reviews in Plant Science*) — *G. arboreum* salinity tolerance physiology
- Francois, Grieve, Maas & Epstein (1994, *Agronomy Journal*) — fiber properties under salt stress
- Sharif, Raza, Ahmad et al. (2019, *Journal of Plant Growth Regulation*) — OAM for diploid cotton
- SRDI (2020) — Noakhali district coastal salinity survey data
